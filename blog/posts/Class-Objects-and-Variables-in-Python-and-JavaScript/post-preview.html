Yesterday findings on globals() and locals() function brought me to how objects are treated in a function or other words passing objects to functions.

What seems to be common is that primitive types are passed by value and reference types are passed by reference. There are lots of differences under the hood (like passing by assignment)in languages that I searched for (JavaScript and Python) but I find Python amazingly clear on data types (mutable, Immutable, subscriptable, etc).
